# Task Summary: 分类管理长按交互优化

## 优化内容

### 长按交互改进
- **修改前**：长按弹出 DropdownMenu 菜单
- **修改后**：长按直接在卡片右侧显示删除和取消按钮，松开自动隐藏

### 按钮布局优化
- 按钮偏移右侧：增加 16dp Spacer
- 按钮间距：8dp（从 4dp 增加）
- 仅显示图标：移除所有文字标签，使用 contentDescription

### 状态切换
```
常态：       [🍜 餐饮]            [➕] [✏️]
长按时：     [🍜 餐饮]            [🗑️] [❌]
拖动时：     [🍜 餐饮]    拖动中...
```

### 技术细节
- 新增 `showDeleteButtons` 状态控制删除按钮显示
- `onLongClick` 设置 `showDeleteButtons = true`
- `onDragEnd`/`onDragCancel` 设置 `showDeleteButtons = false`
- 移除 DropdownMenu，直接使用 IconButton
