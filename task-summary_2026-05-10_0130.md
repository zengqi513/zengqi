## 讯飞SDK下载步骤

### 1. 下载SDK
访问 https://www.xfyun.cn/sdk/dispatcher
- 选择"语音听写" → Android SDK
- 下载SDK包

### 2. 放入项目
```
app/
├── libs/
│   └── Msc.jar              ← 放入这里
└── src/main/jniLibs/
    ├── arm64-v8a/
    │   └── libmsc.so        ← 放入这里
    ├── armeabi-v7a/
    │   └── libmsc.so        ← 放入这里
    ├── x86/
    │   └── libmsc.so        ← 放入这里
    └── x86_64/
        └── libmsc.so        ← 放入这里
```

### 3. 启用SDK
修改 `app/build.gradle.kts`：
```kotlin
// 取消这行的注释
implementation(files("libs/Msc.jar"))
```

### 4. 替换实现
修改 `XunfeiSpeechRecognizer.kt`，将占位符替换为真实SDK调用（代码已准备好，只需取消注释）。
