# Task Summary: 银行收入通知被误排除修复

## 问题
工商银行收入通知被错误标记为"已排除"：
```
工商银行07:52:10动账通知|尾号8166卡5月2日07:52收入(微信零钱提现财付通)0.02元。请点击查看详情。|尾号8166结果:已排除
```

## 根因
v4 添加的 `点击查看` 排除关键词误杀了银行通知（银行通知通常包含"请点击查看详情"）。

## 修复方案
将排除词分为两类：
1. **EXCLUDE_KEYWORDS_STRICT** - 严格排除（所有来源都排除）
   - 能量/物流/验证码/账单提醒等明确非交易
2. **EXCLUDE_KEYWORDS_MARKETING** - 营销广告（仅非银行/支付来源排除）
   - 1分钱/秒杀/免费领/点击查看等

银行/支付宝/微信/电商类通知：
- ✅ 只检查严格排除词
- ❌ 不检查营销广告词（避免误杀）

其他来源：
- ✅ 检查严格排除词
- ✅ 检查营销广告词

## 验证
工商银行通知含"点击查看"但不会被排除，正常记录为收入。
