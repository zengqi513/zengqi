## 优化内容

### 1. 语音识别失败重试机制

**自动重试**:
- 识别失败时自动重试最多2次
- 每次重试间隔1秒
- 3次失败后才显示错误提示

**手动重试**:
- 显示错误图标和提示
- 提供"重试"按钮手动触发

### 2. 方言/口音支持

**方言选择器**:
```kotlin
方言：普通话 ▼
      ├── 普通话
      ├── 粤语
      ├── 四川话
      ├── 东北话
      ├── 河南话
      └── 湖南话
```

**注意**: 方言选择UI已添加，实际识别效果取决于设备语音识别服务的能力

### 3. 语音指令快捷操作

**增强解析引擎**:

## 技术实现

### 新增状态管理
```kotlin

var showRetry by remember { mutableStateOf(false) }
var recognitionError by remember { mutableStateOf<String?>(null) }

// 方言选择
var selectedDialect by remember { mutableStateOf("普通话") }
val dialects = listOf("普通话", "粤语", "四川话", "东北话", "河南话", "湖南话")
```

### ViewModel扩展
```kotlin
private val _voiceRecognitionError = MutableStateFlow<String?>(null)
val voiceRecognitionError: StateFlow<String?> = _voiceRecognitionError.asStateFlow()

fun onVoiceRecognitionError(error: String)

```
