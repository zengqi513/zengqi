# Task Summary: 分类管理界面优化

## 优化内容

### 长按菜单简化
- **修改前**：编辑、添加子分类、删除
- **修改后**：仅保留"删除"（"取消"由系统自动提供）

### 右侧按钮添加
在一级分类卡片右侧添加两个操作按钮（非拖动状态下显示）：
- ➕ **添加子分类** - 添加当前分类的子分类
- ✏️ **修改** - 修改当前分类的名称和图标
