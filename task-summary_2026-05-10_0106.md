## 优化内容

### 加密备份密码对话框

**问题**: 加密备份使用硬编码密码"123456"，存在安全隐患

**解决方案**: 添加密码输入对话框，让用户自定义加密密码

### 对话框功能

| 功能 | 说明 |
|------|------|
| 密码输入 | 支持密码隐藏显示（PasswordVisualTransformation） |
| 密码确认 | 两次输入验证，防止输入错误 |
| 密码强度 | 最少6位字符要求 |
| 错误提示 | 密码不一致或长度不足时显示红色提示 |
| 取消操作 | 备份过程中可取消（按钮禁用状态） |

### UI设计

```
┌─────────────────────────────────────┐
│  🔒 设置备份密码                     │
│                                     │
│  请设置加密密码（至少6位），密码     │
│  用于保护您的备份数据                 │
│                                     │
│  ┌─────────────────────────────────┐│
│  │ 🔑 密码              [••••••]   ││
│  └─────────────────────────────────┘│
│                                     │
│  ┌─────────────────────────────────┐│
│  │ 🔑 确认密码          [••••••]   ││
│  └─────────────────────────────────┘│
│                                     │
│  [取消]                    [确认]   │
└─────────────────────────────────────┘
```

### 验证规则

1. **长度验证**: 密码至少6位，不足时提示"密码至少需要6位"
2. **一致性验证**: 两次输入必须一致，不一致时提示"两次输入的密码不一致"
3. **实时清除**: 输入时自动清除错误提示

### 交互流程

```
用户点击"加密备份"按钮
        ↓
显示密码对话框（密码、确认密码输入框）
        ↓
用户输入密码 → 实时验证
        ↓
点击确认 → 验证通过 → 关闭对话框 → 开始备份
        ↓
备份完成 → 分享加密备份文件
```
