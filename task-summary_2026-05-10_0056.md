## 优化内容

### 1. 备份加密

**EncryptedBackupHelper.kt** - 新增加密备份支持

| 特性 | 实现 |
|------|------|
| 加密算法 | AES-256-GCM |
| 密钥派生 | PBKDF2WithHmacSHA256 (10万次迭代) |
| 盐值 | 16字节随机 |
| IV | 12字节随机 |
| 文件扩展名 | `.abk` (AutoBookkeeper Backup) |

**API**:
```kotlin
// 创建加密备份
EncryptedBackupHelper.createEncryptedBackup(context, db, password)

// 恢复加密备份
EncryptedBackupHelper.restoreEncryptedBackup(context, db, file, password)

// 验证密码
EncryptedBackupHelper.verifyPassword(file, password)
```

### 2. 自动云备份

**CloudBackupManager.kt** - 新增云备份管理

| 功能 | 说明 |
|------|------|
| 自动备份间隔 | 24小时 |
| 本地备份保留 | 最近10个 |
| 云存储支持 | Google Drive, OneDrive, Dropbox |
| 备份历史 | 查看所有备份记录 |

**支持的云存储**:
- `google_drive` - Google Drive
- `onedrive` - Microsoft OneDrive  
- `dropbox` - Dropbox

**自动备份流程**:
1. 检查上次备份时间
2. 创建加密/普通备份
3. 清理旧备份
4. 上传到云存储（通过系统分享）
5. 更新备份时间

### 3. 用户偏好设置扩展

**UserPreferences.kt** 新增云备份设置:

```kotlin
data class CloudBackupSettings(
    val enabled: Boolean,           // 是否启用自动备份
    val encrypt: Boolean,           // 是否加密
    val encryptionPassword: String, // 加密密码
    val cloudProvider: String?,     // 云存储提供商
    val lastBackupTime: Long        // 上次备份时间
)
```

**设置方法**:
- `setCloudBackupEnabled(enabled)`
- `setCloudBackupEncrypt(encrypt, password)`
- `setCloudBackupProvider(provider)`
- `updateLastBackupTime(timeMillis)`

## 使用示例

### 创建加密备份
```kotlin
val manager = CloudBackupManager(context)
val result = manager.performBackup(
    encrypt = true,
    password = "your_password",
    uploadToCloud = true,
    cloudProvider = CloudBackupManager.PROVIDER_GOOGLE_DRIVE
)
```

### 恢复加密备份
```kotlin
val manager = CloudBackupManager(context)
val result = manager.restoreBackup(encryptedFile, "your_password")
```

### 启动自动备份
```kotlin
val manager = CloudBackupManager(context)
manager.startAutoBackup() // 后台协程每24小时检查
```

## 安全说明

1. **密码安全**: 密码使用PBKDF2派生密钥，10万次迭代
2. **数据安全**: AES-256-GCM认证加密，防止篡改
3. **本地存储**: 密码只存储在本地DataStore，不上传
4. **文件识别**: 加密文件使用.abk扩展名，普通备份用.json
